QuantumWalk.execute_all_measured — Method.
execute_all_measured(qwd::QWDynamics{<:QWModelDiscr}, initstate, runtime)

Evolve initstate acording to QWDynamics qwd for time runtime. runtime needs to be nonnegative. Quantum walk model needs to be discrete. As a result return matrix of type Matrix{Float64} for which i-th column is measurement probability distribution in i-1-th step.

julia> qwe = QWEvolution(Szegedy(CompleteGraph(4)));

julia> initstate = spzeros(16); initstate[1] = initstate[2] = 1/sqrt(2.);

julia> execute_all_measured(qwe, initstate, 2)
4×3 Array{Float64,2}:
 1.0  0.555556  0.69273
 0.0  0.197531  0.0609663
 0.0  0.123457  0.123152
 0.0  0.123457  0.123152
